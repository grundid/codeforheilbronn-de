CFH-Badges

Die Badges werden an alle Mitglieder von code for heilbronn ausgegeben. An einen Badge sind immer einige Anforderungen gekoppelt. An den Badges kann jeder sehen, wen er zu einem bestimmten Thema fragen kann. Außerdem sind sie Anerkennung für übernommene Aufgaben.

Auf der Seite des Badges sind alle Personen aufgelistet, die den Badge bereits erworben haben. Grundsätzlich kann jeder nur einen neuen Badge pro Monat erhalten.
